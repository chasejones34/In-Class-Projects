Welcome to the RNG Baseball Game!
All games are 3 innings
There are two game modes, single player or multiplayer
The user is prompted to input either "single" for single player, or "multi" for multiplayer
In single player, the user must guess within a certain range of a randomly generated number
The tables look like so:
    single if difference is between 30 and 20
    double if difference is between 20 and 10
    triple if difference is between 10 and 5
    home run if difference is between 5 and 0
In multiplayer, the user will play agains another person, with player 1 inputing a pitch number
    and player 2 inputing a swing number until player 1 gets 3 outs. switching sides
